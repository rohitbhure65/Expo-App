1. General Requirements

The application will be developed using React Native with Expo.

The app will support Android and iOS platforms.

The user interface should be simple, clean, and responsive.

2. User Features

Users can view a list of products.

Users can view product details (image, name, price, description).

Users can add products to the cart.

Users can remove products from the cart.

Users can view the total price in the cart.

3. Product Management

Products will be displayed using dummy data or static API.

Each product will include:

Product image

Product name

Price

Short description

4. Cart Functionality

Add to cart functionality.

Update quantity of products in the cart.

Remove products from the cart.

Display total items and total cost.

5. Navigation

Bottom tab navigation (Home, Cart, Profile).

Smooth screen transitions.

6. Non-Functional Requirements

Fast loading performance.

Reusable and well-structured components.

Proper error handling for empty states (no products, empty cart).

Basic state management (Context API or simple state).
